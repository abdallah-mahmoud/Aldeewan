# الدِّيوانُ (AlDeewan)

دفتر ديونك ومصاريفك... صار في جيبك.

---

## 📖 عن التطبيق

**الدِّيوانُ** هو تطبيق ويب بسيط ومصمم بعناية لأصحاب المحلات التجارية الصغيرة والبقالات، لمساعدتهم على إدارة الديون والحسابات اليومية بكل سهولة. يركز التطبيق على الوضوح وسهولة الاستخدام، مع واجهة تدعم اللغة العربية بشكل أساسي (من اليمين إلى اليسار) وتصميم عصري ونظيف. يعمل التطبيق بشكل كامل بدون الحاجة للاتصال بالإنترنت، حيث يتم حفظ جميع البيانات بأمان على جهازك.

---

## ✨ الميزات الحالية

- **🏠 الرئيسية:** لوحة تحكم سريعة تعرض ملخصاً مالياً لأهم الأرقام (إجمالي المستحق لك وإجمالي المستحق عليك، دخل ومصروفات الشهر) مع اختصارات لإضافة العمليات اليومية.
- **👥 سجل الديون (الزمم):**
  - إضافة وتعديل العملاء والموردين بسهولة.
  - عرض كشف حساب مفصل لكل شخص مع سجل كامل للمعاملات.
  - تسجيل عمليات الدَين والسداد وتتبع الأرصدة المتأخرة.
  - إمكانية مشاركة كشف الحساب كنص عبر أي تطبيق (واتساب، بريد إلكتروني، ...إلخ).
  - بحث وفلترة متقدمة للوصول السريع.
- **💵 دفتر النقدية:**
  - تسجيل الإيرادات (الدخل) والمصروفات اليومية وتصنيفها.
  - عرض ملخص لإجمالي الدخل والمصروفات وصافي التدفق النقدي للفترة المحددة.
  - فلاتر متقدمة حسب النوع (دخل/مصروف) والفترة الزمنية.
- **📊 التقارير والتصدير:**
  - إنشاء **كشف حساب شخصي** لأي عميل أو مورد خلال فترة زمنية محددة.
  - إنشاء **تقرير التدفق النقدي** الذي يوضح الأرباح والخسائر حسب الفئات.
  - تصدير البيانات إلى ملفات **CSV** لمزيد من التحليل أو الأرشفة.
- **⚙️ الإعدادات وإدارة البيانات:**
  - تخصيص مظهر التطبيق (فاتح، داكن، تلقائي).
  - اختيار العملة والمنطقة الزمنية المناسبة.
  - **النسخ الاحتياطي (Export):** حفظ نسخة كاملة من بياناتك في ملف JSON آمن.
  - **الاستيراد (Import):** استعادة بياناتك من ملف نسخة احتياطية.
- **🚀 بدء سهل (Onboarding):** عند استخدام التطبيق لأول مرة، يمكنك اختيار البدء ببيانات تجريبية لاستكشاف الميزات أو البدء بسجل نظيف.
- **🌍 دعم لغتين:** واجهة مستخدم كاملة باللغة العربية والإنجليزية.
- **🔒 يعمل بدون انترنت:** جميع بياناتك محفوظة محلياً على جهازك ولا تحتاج إلى اتصال بالإنترنت.

## 💻 التقنيات المستخدمة

تم بناء هذا التطبيق باستخدام التقنيات الحديثة التالية:

- **React & TypeScript:** لبناء واجهة مستخدم قوية وموثوقة.
- **Tailwind CSS:** لتصميم واجهة عصرية وسريعة الاستجابة.
- **Dexie.js (IndexedDB):** لحفظ البيانات بشكل دائم وموثوق في متصفحك.
- **Lucide React:** لمجموعة أيقونات نظيفة ومتناسقة.
- **TanStack Virtual:** لتحسين أداء القوائم الطويلة وضمان سلاسة التمرير.

## 📂 بنية المشروع

تم تنظيم ملفات المشروع بشكل يسهل فهمه وتطويره:

```
├── components/ - المكونات القابلة لإعادة الاستخدام (أزرار، نماذج، ...إلخ).
├── context/    - لإدارة الحالة العامة للتطبيق (البيانات، اللغة، المظهر).
├── data/       - يحتوي على البيانات الأولية التجريبية.
├── screens/    - يمثل كل شاشة رئيسية في التطبيق (الرئيسية، السجل، ...إلخ).
├── hooks/      - خطافات مخصصة (Custom Hooks) مثل useDebounce.
├── utils/      - دوال مساعدة (مثل حساب الأرصدة، التصدير، ...إلخ).
├── types.ts    - تعريف أنواع البيانات المستخدمة في التطبيق.
├── db.ts       - تعريف بنية قاعدة البيانات المحلية (IndexedDB).
├── App.tsx     - المكون الرئيسي الذي يجمع كل أجزاء التطبيق.
└── index.tsx   - نقطة دخول التطبيق.
```
