# الدِّيوانُ (AlDeewan)

دفتر ديونك ومصاريفك... صار في جيبك.

---

## 📖 عن التطبيق

**الدِّيوانُ** هو تطبيق ويب بسيط ومصمم بعناية لأصحاب المحلات التجارية الصغيرة والبقالات، لمساعدتهم على إدارة الديون والحسابات اليومية بكل سهولة. يركز التطبيق على الوضوح وسهولة الاستخدام، مع واجهة تدعم اللغة العربية بشكل أساسي (من اليمين إلى اليسار) وتصميم عصري ونظيف. يعمل التطبيق بشكل كامل بدون الحاجة للاتصال بالإنترنت، حيث يتم حفظ جميع البيانات بأمان على جهازك.

---

## ✨ الميزات الأساسية

- **🏠 الرئيسية:** لوحة تحكم سريعة تعرض ملخصاً مالياً لأهم الأرقام (إجمالي المستحق لك وإجمالي المستحق عليك) مع اختصارات لإضافة العمليات اليومية.
- **👥 سجل الديون (الزمم):**
  - إضافة وتعديل العملاء والموردين.
  - عرض كشف حساب مفصل لكل شخص على حدة.
  - تسجيل عمليات الدَين والسداد بسهولة.
- **💵 دفتر النقدية:**
  - تسجيل الإيرادات (الدخل) والمصروفات اليومية.
  - عرض ملخص لإجمالي الدخل والمصروفات وصافي التدفق النقدي.
- **⚙️ الإعدادات:**
  - تخصيص مظهر التطبيق بين الوضع الفاتح، الداكن، أو التلقائي حسب النظام.
  - تغيير عملة التطبيق (ريال سعودي، جنيه سوداني، درهم إماراتي، جنيه مصري).
- **🌍 دعم لغتين:** واجهة مستخدم كاملة باللغة العربية والإنجليزية مع تصميم متوافق مع اتجاه القراءة من اليمين لليسار (RTL).
- **🔒 يعمل بدون انترنت:** جميع بياناتك محفوظة محلياً على جهازك ولا تختفي عند إغلاق التطبيق.

---

## 💻 التقنيات المستخدمة

تم بناء هذا التطبيق باستخدام التقنيات الحديثة التالية:

- **React:** مكتبة JavaScript لبناء واجهات المستخدم.
- **TypeScript:** لإضافة أنواع البيانات وجعل الكود أكثر قوة وموثوقية.
- **Tailwind CSS:** إطار عمل CSS لتصميم واجهة عصرية وسريعة الاستجابة.
- **Dexie.js (IndexedDB):** لحفظ البيانات بشكل دائم وموثوق في متصفحك، مما يتيح للتطبيق العمل بدون انترنت.

## 📂 بنية المشروع

تم تنظيم ملفات المشروع بشكل يسهل فهمه وتطويره:

```
├── components/ - المكونات القابلة لإعادة الاستخدام (مثل الأزرار، النماذج، القوائم).
├── context/    - لإدارة الحالة العامة للتطبيق (مثل بيانات المستخدمين، اللغة، والمظهر).
├── data/       - يحتوي على البيانات الأولية التجريبية.
├── screens/    - يمثل كل شاشة رئيسية في التطبيق (الرئيسية، السجل، النقدية، إلخ).
├── types.ts    - لتعريف أنواع البيانات المستخدمة في التطبيق.
├── db.ts       - لتعريف بنية قاعدة البيانات المحلية.
├── App.tsx     - المكون الرئيسي الذي يجمع كل أجزاء التطبيق.
└── index.tsx   - نقطة دخول التطبيق.
```

## 🚀 خطط مستقبلية

نحن نعمل على إضافة المزيد من الميزات لتحسين التطبيق، وتشمل:

- **📊 التقارير:** إمكانية إنشاء وتصدير تقارير مالية بصيغة PDF أو CSV.
- **📦 إدارة المخزون:** نظام بسيط لتتبع المنتجات والمخزون.
- **☁️ النسخ الاحتياطي والمزامنة:** حفظ البيانات بشكل آمن على السحابة (اختياري).
- **🔔 التنبيهات:** إرسال تذكيرات بمواعيد استحقاق الديون.
