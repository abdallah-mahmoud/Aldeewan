﻿# تطبيق الديوان (Aldeewan)

تطبيق حديث للإدارة المالية والمحاسبة الشخصية والتجارية مصمم لمساعدتك على تتبع الديون المصروفات والإيرادات بسهولة ودقة.

##  مميزات التطبيق
- **لوحة تحكم شاملة:** عرض ملخصات مالية سريعة (دائن مدين رصيد) مع رسوم بيانية توضيحية.
- **دفتر الأستاذ (Ledger):** إدارة حسابات العملاء والموردين وتسجيل عمليات البيع والشراء الآجل.
- **دفتر النقدية (Cashbook):** تتبع حركة النقدية اليومية (إيرادات ومصروفات).
- **تقارير مفصلة:** إمكانية تصدير كشوفات الحساب والتقارير المالية (CSV/PDF).
- **واجهة عصرية:** تصميم "Fintech" حديث يدعم الوضع الليلي واللغة العربية بالكامل (خط Cairo).
- **عملة محلية:** دعم كامل للجنيه السوداني (SDG).

##  التقنيات المستخدمة
- **Framework:** Flutter (Dart)
- **State Management:** Riverpod
- **Database:** Isar (Local NoSQL Database)
- **Navigation:** GoRouter
- **Fonts:** Google Fonts (Cairo)
- **Icons:** Lucide Icons
